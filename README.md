# React-Native-Test
React Native 在windows环境下的环境搭建及测试

克隆项目
https://github.com/nangongxixi/React-Native-Test.git

cd React-Native-Test  cd到项目根目录

npm i        安装依赖

adb devices  检测真机或者模拟器是否正确链接

react-native run-android   编译运行
